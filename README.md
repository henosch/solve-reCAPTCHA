## üì± Installation

### Browser Setup
Install one of these browsers with userscript support:
- **Firefox**: https://play.google.com/store/apps/details?id=org.mozilla.firefox
- **Firefox Nightly**: https://play.google.com/store/apps/details?id=org.mozilla.fenix  
- **Lemur Browser**: https://play.google.com/store/apps/details?id=com.lemurbrowser.exts

### Userscript Manager
Install Tampermonkey from your browser's add-on store.

### Script Installation
1. Download this script
2. Open Tampermonkey Dashboard
3. Click "+" (New script)
4. Copy & paste this script content
5. Replace the API key placeholder
6. Save and install

## üß™ Usage

1. Navigate to any reCAPTCHA page
   - https://www.google.com/recaptcha/api2/demo
   - https://2captcha.com/de/demo/recaptcha-v2
2. Click the **Buster icon** that appears
3. The captcha will be solved automatically

## üîß Troubleshooting

**Icon not showing?**
- Check if Tampermonkey is enabled
- Verify script is installed (green checkmark)
- Reload the page

**Not working?**
- Check browser console for errors
- Verify API key is correct
- Ensure internet connection
